Choisis un mode : Personnalisé (tu règles k & n) ou Niveaux (préconfigurés).
« Générer » crée un k-partite (complet en perso).

« Personnalisé » : k = nombre de parties, n = sommets par partie.
« Niveaux » : graphes préconfigurés (aucune arête dans une même partie).

Opérations rapides

À propos

Ajouter une arrête : Cliquer sur deux nœuds.
Supprimer une arrête : Double-cliquer sur une arrête.
Vous pouvez aussi déplacer les nœuds pour ajuster le graphe. Sélectionnez un nœud pour le déplacer.
Pour en sélectionner plusieurs, utilisez la touche Shift (Majuscule).
Bouton "Réinitialiser" : supprime tous les nœuds et arêtes.
Bouton "adapter à la vue" : recentre et zoom sur le graphe.
Bouton "Relancer le layout" : réorganise le graphe.
